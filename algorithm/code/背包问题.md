# 背包问题

## 0-1背包

- `f[i][j]`表示对于前i个物品，背包容量为j的最大价值是多少
- 以对于第 i 个物品来说，是选还是不选，对此进行划分

```c++
#include <iostream>

const int N = 1010;
int v[N], w[N];
int f[N];
int n, m;

int main() {
    std::cin >> n >> m;
    for (int i = 1; i <= n; i++) {
        std::cin >> v[i] >> w[i];
    }

    // for (int i = 1; i <= n; i++) {
    //     for (int j = 1; j <= m; j++) {
    //         if (j >= v[i]) {
    //             f[i][j] = std::max(f[i - 1][j], f[i-1][j - v[i]] + w[i]);
    //         } else { 
    //             f[i][j] = std::max(f[i][j], f[i - 1][j]);
    //         }
    //     }
    // }
    
    for (int i = 1; i <= n; i++) {
        for (int j = m; j >= v[i]; j--) {
            f[j] = std::max(f[j], f[j - v[i]] + w[i]);
        }
    }

    std::cout << f[m] << std::endl;
}
```
