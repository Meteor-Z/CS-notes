## ssh主机

在windwos上如果如果使用ssh进行联机，那么可以使用xshell（我觉得不错）

- 收发文件
  - `sudo apt install lrzsz`
  - `rz`或者拖动文件即可

## 相关命令

- ctrl + z: 挂起这个程序(这个程序就暂停了)
- jobs: 显示相关程序
- bg(background): 将一个程序切换到后端执行
- fg(foreground): 将程序切换到前端执行

但是这个也不是直接能够挂后台的，可以使用tmux,
